Get Started

The Battle of AI is a community project of the German IT-Youtuber The Morpheus Tutorials. Its purpose is to enhance your coding and AI skills and compete against others.

Start easy. Improve your code. Master the Game.

For the easy start, there are example AIs to get right onto it.

Download the project from Github and dive deep.

  • Learn Python
  • Use the sample code and implement the turn()-function
  • You can find it on Github

Wait? What game do we play?

CoRe - Computer Reversi

Simple concepts, complex strategies.

Place tiles of your color. If you surround the enemies tiles, they are converted and enhance your army.

When all tiles are placed, the player with the most of them wins the game.

Developed by Anthony Cipriano
